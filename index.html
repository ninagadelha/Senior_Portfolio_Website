    Projects

    Bargain Buddy

    Bargain Buddy's target audience is college students, as it combines grocery shopping with social media interactions. As a frontend developer, I contributed to both the app's design and functionality. I implemented API requests (GET, PUT, POST, DEL) through JSON objects and arrays, enabling users to create and delete accounts, manage wishlists, add items to their carts and participate in group-chats with other users. Additionly, I was primarily responsible for designing the app's user interface (UI), emphasizing usability and visual appeal.

    Vulnerability Detection

    This project was completed in my first cybersecurity course and played a key role in sparking my interest in the field. We were tasked with indentifying vulnerabilities and misconfigurations on a virtual machine and describing the appropraite fixes. The project provided hands-on experience navigating systems, detecting poor security practices, and applying concepts learned in lecture to develop practical solutions. Through this project, I strengthed my skills in system analysis, vulnerability assessment, and implementing effective secuirty solutions.

    MIPS Processors

    In Computer Organization and Assembly Level Programming (CPRE 381), I collaborated with another student to develop three versions of a MIPS processor: Single Cycle, Software-Scheduled Pipeline and Hardware-Scheduled Pipeline. We implemented the processors using VHDL and wrote test cases using Assembly. The project began with constructing basic logic gates, which we used to build more complex subcomponents. These were eventually integrated into a fully functioning Single Cycle Processor. We then expanded from that design to create our Software- and Hardware-Scheduled Pipeline Processors.
